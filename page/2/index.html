python监听鼠标键盘操作 2020-07-27|python|hook-pyHook

概述

通过pyHook模块来实现监听鼠标键盘操作

参考链接：https://sourceforge.net/p/pyhook/wiki/PyHook_Tutorial/

为此我们需要安装pyHook模块和他的好基友pythoncom模块。

什么是hook？

hook就是钩子，在windows发生了事件（事件就是消息）时触发执行相应的逻辑，当然这个逻辑需要由你自己去写。hook是一种编程思想，与语言、系统无关。

windows hook思想是基于消息的，窗口打开是消息，关闭是消息，鼠标点击是消息，键盘输入是消息，等等等等……

就像vue.js中的钩子函数：watch：当页面中的某个值发生改变的时候触发；beforeMount：页面元素加载完前执行，mounted：页面元素加载完成后执行相应的逻辑

而pyHook包装了windows底层的API，为特定的windows事件提供了挂载钩子，所以通过引入pyHook模块，就可以实现对windows事件的监听，从而实现自己想要达到的目的，如：鼠标点击事件、键盘输入事件等。

想要使用pyHool，我们还得引入pythoncom模块，通过pythoncom我们可以调用windows提供的组件功能。

redis的一些常见问题 2020-04-17|javaredis|redis-缓存

1. 简单介绍一下 Redis 呗!

简单来说 Redis 就是一个使用 C 语言开发的数据库，不过与传统数据库不同的是 Redis 的数据是存在内存中的 ，也就是它是内存数据库，所以读写速度非常快，因此 Redis 被广泛应用于缓存方向。

另外，Redis 除了做缓存之外，Redis 也经常用来做分布式锁，甚至是消息队列。

Redis 提供了多种数据类型来支持不同的业务场景。Redis 还支持事务 、持久化、Lua 脚本、多种集群方案。

2. 分布式缓存常见的技术选型方案有哪些？

分布式缓存的话，使用的比较多的主要是 Memcached 和 Redis。不过，现在基本没有看过还有项目使用 Memcached 来做缓存，都是直接用 Redis。

Memcached 是分布式缓存最开始兴起的那会，比较常用的。后来，随着 Redis 的发展，大家慢慢都转而使用更加强大的 Redis 了。

分布式缓存主要解决的是单机缓存的容量受服务器限制并且无法保存通用的信息。因为，本地缓存只在当前服务里有效，比如如果你部署了两个相同的服务，他们两者之间的缓存数据是无法共同的。
