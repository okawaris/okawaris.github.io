python程序打包成exe
2020-07-28|python

Python是一种脚本语言，也就是解释型的语言，需要解释器来进行解释以后才可以执行，而Python源代码需要在Python虚拟机上面运行，但是我们做好的程序，不可能给用户使用的时候还让他安装一个Python环境，所以今天我们讲解如何将Python程序打包为exe可执行文件。

一、通过pyinsatller来打包

安装pyinsatller

会有以下提示

Successfully installed altgraph-0.17 future-0.18.2 pefile-2019.4.18 pyinstaller-3.6 pywin32-ctypes-0.2.0

表示已经安装成功

打包python程序

  1. 打开cmd窗口进入你想要打包的python的文件夹

  2. 执行命令

    filename表示你的Python程序文件名

    -w 表示隐藏程序运行时的命令行窗口（不加-w会有黑色窗口）

    括号内的为可选参数，-i icofile表示给程序加上图标，图标必须为.ico格式。

    icofile表示图标的位置，建议直接放在程序文件夹里面，这样子打包的时候直接写文件名就好。

    输入完成，按回车，就会开始自动打包了，第一次打包过程可能比较缓慢。

    实际例子：

    toq是我的图标文件，由于我跟py程序放在同一个文件夹下，所以这里只需打出文件名字即可。

    当输出以下信息时即表示打包成功

    8729 INFO: Building EXE from EXE-00.toc completed successfully.

    生成的应用程序文件，在当前目录下的\dist文件夹下

    打开windows任务管理器，就可以看到自己应用程序的进程，如下图

    该执行程序可以脱离python运行环境依赖，发给任何人的电脑使用。
