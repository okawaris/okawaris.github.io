python模拟键鼠操作
2020-07-23|python

一、前言

通过python来模拟键鼠操作有许多种方法，但是个人最好用的还是pymouse模块和pykeyborad模块。

目前pymouse和pykeyborad都已集成到PyUserInput里面了，所以现在可以只安装PyUserInput就可以了

安装PyUserInput需要先安装pyHook（注意要兼容python3）

二、安装

1、安装python

  1. 下载pythonhttps://www.python.org/

  2. 打开CMD命令窗口，输入py，后能正确显示python版本就表示安装成功了。

2、安装PyUserInput

2.1、安装pyHook

2.1.1、下载pyhook_py3k

地址：https://github.com/Answeror/pyhook_py3k

2.1.2、下载swig

地址：http://www.swig.org/download.html

点击后，会自动进入下载页面。将下载下来的文件放到D:\temp文件下，并解压

2.1.3、编译

  1. 解压缩pyhook_py3k

  2. 通过控制台cmd进入pyhook_py3k所在目录下

    • 键盘同时按下win+r键，并输入cmd,回车打开命令窗口


    • 输入**d:**并回车进入D盘根目录

    • 输入cd \temp\pyhook_py3k-master并回车进入pyhook_py3k所在目录

  3. 运行命令:

链接: https://pan.baidu.com/s/1GmYMp8cRY7gsuKi-Ccb0ng

提取码: 353g

2.1.4、安装

编译完成后,在该目录下运行命令: pip install .

但是我安装时报错了

意思是说我的pip版本是20.0.2，太低了，需要去将版本提到20.1.1，我们就按照提示去升级即可。

我们执行 python -m pip install –upgrade pip

然后继续运行命令:pip install .

至此，pyHook的安装就完成了。

2.2、安装PyUserInput

运行命令：pip install PyUserInput

三、使用

引入

实例化:

鼠标：
m.click(x,y,button,n) –鼠标点击
x,y –是坐标位置
buttong –1表示左键，2表示点击右键
n –点击次数，默认是1次，2表示双击

m.move(x,y) –鼠标移动到坐标(x,y)

x_dim, y_dim = m.screen_size() –获得屏幕尺寸

键盘：

k.type_string(‘Hello, World!’) –模拟键盘输入字符串
k.press_key(‘H’) –模拟键盘按H键
k.release_key(‘H’) –模拟键盘松开H键
k.tap_key(“H”) –模拟点击H键
k.tap_key(‘H’,n=2,interval=5) –模拟点击H键，2次，每次间隔5秒
k.tap_key(k.function_keys[5]) –点击功能键F5
k.tap_key(k.numpad_keys[5],3) –点击小键盘5,3次

组合键:
例如同时按alt+tab键盘
k.press_key(k.alt_key) –按住alt键
k.tap_key(k.tab_key) –点击tab键
k.release_key(k.alt_key) –松开alt键
