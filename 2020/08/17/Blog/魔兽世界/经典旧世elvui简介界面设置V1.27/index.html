食用方法

方法一： 插件包解压使用

  • 删除_classic_里的interface、WTF、interface.obsoleted、大脚

  • 将下载好的Interface扔进_classic_里

  • 登陆游戏ESC-ElvUI-配置文件-导入ElvUi字符串

  • 修改完聊天框输入/rl

  • 出现所有弹窗一律跳过

  • 伤害字体设置：elvui-一般-材质-战斗文字字体-选伤害字体-然后重启游戏

  • 动作条美化：elvui-动作条-一般-最下面全打勾，重载即可

方法二： 桃乐豆分享配置码直接下载配置，分享码使用方法：

  • 下载安装桃乐豆
  • 选择上方的配置分享
  • 在输入框内粘贴分享码搜索
  • 下载配置即可
  • 分辨率不是1080的重载完记得去elvui-一般-点击一下自动缩放

常用命令宏

Details观察队友装备显示天赋怎么关闭？(聊天框输入命令后/rl)
/run Details.disable_talent_feature = true
姓名版间距
/run SetCVar(“nameplateOverlapH”, 0.8)
/run SetCVar(“nameplateOverlapV”, 1.1)
检查姓名版间距命令
/dump GetCVar(“nameplateOverlapH”)
/dump GetCVar(“nameplateOverlapV”)
伤害数字大小调整
/console WorldTextScale 2
聊天界面职业颜色
/console SET chatClassColorOverride “0”
关闭全屏幕泛光
/console ffxGlow 0
关闭死亡效果
/console ffxDeath 0

问题集锦

Q：间歇性卡顿现象原因
A：ClassicCodex任务标记插件导致，如果接到世界掉落或者全地图掉落的任务请在L任务面板里关闭显示 就不会卡顿了
Q：已经删除的插件为什么在桃乐豆更新软件上还显示？
A：文件夹直接删除插件，在桃乐豆软件不会把它直接删除，而是会提示，目的在于帮助检测本地插件的手工改动，你看到提示后可自行决定是要删除它还是重装它。
Q：插件怎么更新？
A：帖子插件更新请按照食用方法重新操作，单体插件自己用更新器更新
Q：如何整体界面字体大小调整？(不包括单位框体，单位框体需要单独调整)
A：ElvUI-一般—字体大小(根据需求调正)
Q：ElvUI自带 连击点 启用or关闭？
A：ElvUI-单位框体-玩家框体-职业特有条开启or关闭
Q：战场弹窗点击加入无效?
A：1 /rl 后点击加入 2禁用Details 右键小地图战场图标进入
Q：如何加入世界频道?
A：聊天框输入：/join 大脚世界频道 (输入完命令后，右键聊天按钮 退出/加入世界频道)
Q：自动交接任务如何关闭
A：1.界面-插件-ClassicCodex-去掉第一行和第二行的勾 2.ESC-界面-插件-Guidelime-下拉取消自动交接任务(插件包配置两个都是默认关闭的)
Q：DBM打开配置界面后缺失很多选项
A：Klixui-功能美化-美化皮肤里关闭DBM和WA的
Q：单位框体名字血量字体大小如何调整？
A：ElvUI-单位框体-玩家框体-自定义-选中要调节的选项，例如名字：name -字号以及字体，以此类推
Q：看不到拾取框怎么办？
A：禁用快速拾取插件或者关闭自动拾取
Q：如何关闭暴雪自带任务追踪？
A：ESC-界面-显示-自动任务追踪关闭 已经显示在界面的 按L打开任务框体按住shift点击相应任务取消追踪
Q：为什么20码才看见怪的姓名版？
A：暴雪锁定了目标距离不能调节姓名版40码可见
Q：为什么姓名版显示两条亮条？
A：请关闭elvui姓名版
Q：屏幕中间弹出lua报错窗口怎么屏蔽？
A：ESC-界面-插件-AdvancedInterfaceOptions-显示lua错误勾去掉
Q：游戏伤害数字如何调出？
A：ESC-界面-战斗-目标伤害显示勾上 周期性伤害 宠物伤害都勾上
